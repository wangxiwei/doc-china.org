使用Express

使用Express

Express堪称是Node.js领域最为流行的Web开发框架，由著名的开发者TJ开发，现在已经衍生到4.x版本。

Express的"Hello World"

以下是一个简单的使用Express作为服务器的代码，通过res.send()方法将Hello World字符串作为Response返回：

var express = require('express');
var app = express();

app.get('/', function (req, res) {
    res.send('Hello World');
});

app.listen(3000, function () {
    console.log('Express server started.');
});

运行app.js:

node app.js

Express应用生成器

Express亦提供了诸如其他著名Web框架一样的快速生成项目的工具：Express Generator

  • 安装：
npm install -g express-generator
  • 快速生成Express项目：
express test

随后进入到该目录，并安装所依赖的模块：

cd test
npm install

此时，已经可以通过npm start启动这个自动生成的项目了。
